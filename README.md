Welcome to the **Ultimate OSINT Roadmap**. Master the art of Open Source Intelligence—finding information from publicly available sources.

| Level | Core Skills | Primary Goal |
| :--- | :--- | :--- |
| **Entry Level** | Google Dorking, Social Media Investigation, Basic Tools | Find publicly available personal or corporate info |

| **Professional Level** | Automation (Python), Advanced Metadata Analysis | Large-scale intelligence gathering and analysis |
| **Expert Level** | Advanced GEOINT, Intelligence Reporting, OpSec | Master-level strategic intelligence operations |
